括号自动补全

```
inoremap ( ()<ESC>i
inoremap ) <c-r>=ClosePair(')')<CR>
inoremap { {<CR>}<ESC>O
inoremap } <c-r>=ClosePair('}')<CR>
inoremap [ []<ESC>i
inoremap ] <c-r>=ClosePair(']')<CR>
function! ClosePair(char)
    if getline('.')[col('.') - 1] == a:char
        return "\<Right>"
    else
        return a:char
    endif
endfunction
```

编译及调试

```
map <F5> :call CR()<CR>
func! CR()
exec "w"
exec "!g++ -std=c++11 -Wall -Wextra % -o %<"
exec "!./%<"
endfunc
  
map <F6> :call Rungdb()<CR>  
func! Rungdb()  
    exec "w"  
    exec "!g++ -std=c++11 -Wall -Wextra % -g -o %<"  
    exec "!gdb ./%<"  
endfunc
```
